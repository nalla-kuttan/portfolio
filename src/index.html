Fragments Microservice will handle small text and image fragments. It supports multiple formats (text, JSON, images) from various sources like IoT devices and mobile apps. The service will connect seamlessly with existing systems, using HTTP REST API for CRUD operations, format conversions, and secure data storage. It requires GitHub integration for automatic building, testing, and AWS deployment. The system must scale to handle large data volumes and ensure proper authorization for data isolation.
